## MODIFIED Requirements

### Requirement: Device & Hardware Card Field Order

The Device & Hardware card SHALL display fields in a specific order prioritizing core hardware information.

#### Scenario: Hardware card displays fields in priority order
- **WHEN** viewing the Device & Hardware card
- **THEN** the fields are displayed in the following order:
  1. CPU Model
  2. Processor Count
  3. Total RAM
  4. Processor Architecture
  5. OS Architecture
  6. 64-bit Process
  7. Running Under Emulation
